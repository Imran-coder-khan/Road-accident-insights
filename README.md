🚦 **Road Accident Analysis Dashboard | Microsoft Excel Project**

I’m excited to share my latest **Road Accident Analysis Dashboard** developed using **Microsoft Excel**.

This dashboard provides a comprehensive analysis of road accident data, helping to identify patterns related to accident severity, vehicle involvement, road conditions, and environmental factors.

📊 **Key Highlights:**
✅ Total Casualties KPI Monitoring
✅ Fatal, Serious & Slight Severity Analysis
✅ Month-wise Casualties Trend
✅ Vehicle-wise Accident Distribution
✅ Urban vs Rural Comparison
✅ Road Type & Weather Condition Insights
✅ Light vs Dark Accident Analysis

The dashboard is fully interactive using slicers, enabling dynamic filtering and deeper data exploration for better decision-making.

🛠 **Tools Used:**
Microsoft Excel | Pivot Tables | Pivot Charts | Data Cleaning | Dashboard Design | Data Visualization
